
Sprint 7 — API-тесты

Автоматизированные тесты для API-сервиса, выполненные в рамках учебного проекта Яндекс Практикума.

Проект содержит автотесты для сервиса заказов и курьеров:

— создание курьера  
— авторизация курьера  
— создание заказа  
— получение списка заказов

Тесты реализованы с использованием Java, Maven, JUnit4, RestAssured и Allure.

| технология          | версия |
|---------------------|--------|
| Java                | 11     |
| Maven               |        |
| REST Assured        | 5.5.6  |
| Allure Framework    | 2.15.0 |
| JUnit               | 4.13.2 |
| Gson                | 2.13.2 |
| AspectJ Weaver      | 1.9.21 |
| Allure Maven Plugin | 2.10.0 |


Запустить все тесты можно с помощью команды: mvn clean test

Сформировать и открыть отчёт после запуска тестов: allure serve target/allure-results
